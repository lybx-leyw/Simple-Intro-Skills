# Changelog

## 1.0.0 (2026-05-31)

- Initial release of the Quality Trilogy
- Intro: evidence-based self-assessment framework (6 types, weighted dimensions)
- IoE: If-or-Else iterative refinement protocol (based on Li et al. arXiv:2402.12563)
- iloop: autonomous execute → self-assess → decide loop
- Bilingual documentation (English + Chinese), verified examples, CI setup
